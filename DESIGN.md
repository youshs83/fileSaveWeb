# fileSaveWeb 설계 문서

## 1. 프로젝트 개요

### 1.1 프로젝트명
- Phone 아카이빙

### 1.2 목적
- 폰에 쌓여서 누적되어 있는 사진파일을 개인 PC 지정 폴더로 업로드하여 옮길수 있고 옮겨놓은 사진을 대상으로 촬영된 날짜별 웹에서 다시 확인, 미리보기 등의 기능을 지원하고 다시 다운로드도 가능한 웹 앱을 만들고자함.

### 1.3 기술 스택
- Frontend : React.js
- Backend : python
- Database : MariaDB
- api 통신 : RestAPI 혹은 추천
- 상태관리 : Redux, Zustand, Context API 등 추천
- 스타일링 : CSS Modules, Tailwind, styled-components 등을 참고
- 빌드 도구 : Vite, CRA 등 배포가 간단한 것으로 추천

## 2. 화면 구성

### 2.1 화면 목록
| No | 화면명 | 경로(Route) | 설명 | 인증 필요 |
|----|--------|-------------|------|-----------|
| 1 | 로그인 화면| /login | ID/PASSWORD를 입력하는 화면 | Y |
| 2 | 메인 화면| /main | 메인 메뉴와 업로드된 파일의 목록 및 미리보기를 출력하는 화면 | N |
| 3 | 검색 화면| /search | 업로드 된 사진의 정보를 검색하는 화면 | N |
| 4 | 설정 화면| /setting | 사용자정보와 업로드설정 정보를 등록,수정,삭제 하는 관리 화면 | N |

### 2.2 화면 상세

#### P-1. (로그인 화면)
- 경로: /login
- 레이아웃: 가운데 영역에 id, password를 입력하는 input ui 위치
- 주요 기능:
  - id, password를 입력하는 input창을 구현
  - [확인] 버튼을 구현하여, 클릭 시 메인화면으로 이동

#### P-2. (메인 화면)
- 경로: /main
- 레이아웃:
  - 화면 상단에 hearder 영역을 할당하고, native 메뉴바를 이용하여 [검색], [설정] 버튼을 왼쪽정렬하고 [로그아웃]버튼을 오른쪽 정렬하여 위치(해당 hearder는 어떤 화면으로 이동하든지 유지)
  - body 오른쪽 영역에서는 1열은 [파일 업로드] 버튼을 구현하고, 클릭 시 로컬에 있는 파일을 선택하여 업로드 하는 기능
  - body 오른쪽 영역에서는 2열은 업로드 된 파일의 목록을 화면상에서 확인 할 수 있는 영역을 구현
  - body의 왼쪽 영역에서는 영역을 알맞게 배분하여, 최근에 등록 된 사진의 미리보기를 출력하는 화면을 구현 (미리보기 개수는 설정메뉴에서 입력/저장 할수 있도록 구현) - 사진 별 하단 업로드 일자 표기
  - [설정] 버튼을 클릭 시, 설정 화면으로 이동
- 주요 기능:
  - 파일 업로드의 경우, 실제 설정 화면에서 작성된 업로드 경로에 맞춰 server가 실행 된 환경의 실제 경로에 파일이 적재되도록 구현
  - [검색] 버튼을 클릭 시, 검색 화면으로 이동
  - [설정] 버튼을 클릭 시, 설정 화면으로 이동

#### P-3. (검색 화면)
- 경로: /search
- 레이아웃:
  - body 상단부에 검색 영역을 구성
  - body 하단부에 검색 된 결과를 테이블로 출력
- 주요 기능:
  - 검색 영역에 포함 될 대상은 업로드 날짜, 촬영 일자, 사용자명, 파일명으로 구성
  - 검색 영역의 업로드 날짜는 검색시작일, 검색종료일 2개 입력(datepicker)
  - 검색 영역의 촬영일자 날짜는 검색시작일, 검색종료일 2개 입력(datepicker)
  - 검색 영역의 사용자명은 등록되어 있는 사용자 목록 선택(selectbox)
  - 검색 영역의 파일명은 등록되어 있는 파일의 이름이 포함된 대상 검색 (inputbox)

#### P-4. (설정 화면)
- 경로: /setting
- 레이아웃:
  - body 영역에 tab(bootstrap tabs)으로 구분하여 설정할수 있는 화면으로 구성
  - tab은 [사용자정보], [업로드설정]으로 구분
- 주요 기능:
  - [사용자정보] 탭에서는 사용자 정보를 등록,수정,삭제하는 버튼 및 기능을 구현
  - [사용자정보] 탭에서는 등록 된 사용자 정보를 확인할 수 있는 테이블 목록을 출력
  - [업로드설정] 에서는 메인 화면상에서 파일을 업로드할때, 실제 파일을 저장할 로컬 경로를 입력할 수 있게 구현

						-> (설정 화면)
